## Install

1. Clone the repo
   ```sh
   git clone https://github.com/HiImHibiki/ExactCourse.git
   ```
2. `cd` into repo and install NPM packages
   ```sh
   cd ExactCourse
   npm install
   ```
3. Copy `.env.example` to `.env` and fill in your credentials

## Usage

- Backend
  ```sh
  npm run server
  ```
- Frontend
  ```sh
  npm run client
  ```
- Backend & Frontend
  ```sh
  npm run dev
  ```
